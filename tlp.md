## Install TLP

Advanced power management for Linux.

https://linrunner.de/tlp/introduction.html
https://linrunner.de/tlp/installation/ubuntu.html

### T450s

```bash
sudo apt-get update
sudo apt-get install tlp acpi-call-dkms
```

### X60

```bash
sudo apt-get update
sudo apt-get install tlp tp-smapi-dkms
```
